# Dojo Uniq

## Dojo
    
There is an array with some numbers. All numbers are equal except for one. Try to find it!
If no exception, return null;

## Exemples : 

    Entrée [5, 5, 5, 7, 5]
    Retour : 7
 
    Entrée [123, 123, 123, 4]
    Retours : 4
    
    Entrée [6, 6, 6, 6]
    Retours : null
